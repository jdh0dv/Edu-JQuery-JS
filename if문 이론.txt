if(조건문){
 true일 경우의 실행문
}else{
 false일 경우의 실행문
}

if(조건문1){
 조건문1이 true일 경우의 실행문
} else if(조건문2){
 조건문1이 false, 조건문2가 true일 경우의 실행문
} else {
 모든 조건이 false일 경우의 실행문
}



*조건문의 경우 연산기호가 들어감

[연산기호]
> : 크다, 초과
< : 작다, 미만
>= : 크거나 같다, 이상
<= : 작거나 같다, 이하
== : 같다
!= : 다르다

[등호]
= : 오른쪽에 있는 것을 왼쪽에 치환하여라 (변수입력시)
== : 같다 '10' == 10 (true : 데이터형식 상관없이 결과값이 같을 때)
=== : 같다 '10' === 10 (false : 데이터형식까지 같아야 true)

[and or 조건]
A && B : and - 조건이 모두 만족해야 실행
A || B : or - 하나만 만족해도 실행
